Bio

I'm a Python and Bash developer based in Lahore, Pakistan, focused on building automation systems and AI-powered tooling. My work spans from small shell scripts that save minutes a day to full AI agent pipelines that handle complex document processing.

I work heavily with the modern AI stack — OpenAI, LangChain, vector databases, RAG systems — and pair it with solid backend engineering in FastAPI, PostgreSQL, and Supabase. Linux is my home turf; I've been writing Bash since before I touched Python.

If it's repetitive, I want to automate it. If it's text-heavy, I want to put an LLM on it. If it ships, I want a CI pipeline for it.
